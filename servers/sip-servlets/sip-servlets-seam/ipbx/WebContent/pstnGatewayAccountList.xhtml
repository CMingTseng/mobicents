Here you can configure PSTN Gateway account to be used when the PBX needs to dial a PSTN phone number. Accounts are set up in the usual way, you user name, password and gateway host name from your provider. If you are behind a NAT and you want to use PSTN gateway functions, you must configure STUN.See here how to configure STUN with Mobicents Sip Servlets.
