The purpose of this project is to develop a SIP Servlet JSR 289 Spec Implementation on top of Tomcat and to develop interoperability standards between SIP Servlet and JAIN-SLEE so that applications may exploit the strengths of both. The JAIN-SIP implementation from http://jain-sip.dev.java.net will be used as the SIP stack and Mobicents will be used as the SLEE implementation.
