How to run this test:

1 - Compile with 'mvn install'

2 - Run with 'java -jar target/jain-sip-test-0.0.1-SNAPSHOT-with-dependencies.jar'
    This will create some files in the current dir
    
3 - If you don't have sipp, you can run ./download-and-compile-sipp.sh and it will be compiled to your current dir
    You only have to do this once
    
4 - Start sipp using the provided script ./test.sh

5 - Lean back and enjoy. You can stop the test when you want by pressing ctrl^c

6 - A sipp report (a .csv file) will be in the current dir with the results
