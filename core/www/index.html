The Open Source SLEE and SIP Server

Mobicents is a highly scalable event-driven application server with a robust component model and fault tolerant execution environment. Mobicents is the first and only Open Source Platform certified for JSLEE 1.0 compliance. It complements J2EE to enable convergence of voice, video and data in next generation intelligent applications. Web and SIP can be combined together to achieve more sophisticated and natural user experience.

Read quotes from top tier telco operators about Mobicents and IMS.

In addition to telecom, Mobicents is suitable for a variety of problem domains demanding Event Driven Architecture (EDA) for high volume, low latency signaling. Examples include financial trading, online gaming, sensor network integration (RFID) and distributed control.
