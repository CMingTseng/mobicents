This dir contains mobicents app examples. All required dependencies are included in lib/.
Each example has a README.txt file in its top directory with usage instructions.
Some exmaples require ANT 1.7.0
Also see:
http://wiki.java.net/bin/view/Communications/MobicentsExamples

If you have questions, ask on the public user forum:
on forums: http://forums.java.net/jive/category.jspa?categoryID=36 .

If example is part of release use ant bundled with release:
!!! This concerns all ant calls.
/path_to_mobicents/tools/ant/ant -f /path_to_mobicents/examples/example/build.xml deploy
