This is the SIP Resource Adaptor deployment directory

To create the SIP Resource Adaptor deployment jar files, run the 
following:

ant:
ra-deploy
ra-undeploy

output:
ratype/jars/sip-ratype.jar
ratype/events/jars/sip-event.jar
ra/jars/sip-ra.jar

These are resource adaptor type and resource adaptor deployable units

SIP resource adaptor comes with a property file located in the
org/mobicents/slee/resource/sip/sipra.properties

This is an active repository. Consider it unstable.
