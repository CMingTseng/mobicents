7.3.3.1. Introduction to the Mobicents Presence Server

The Mobicents Presence Server is a free and open source implementation of a SIP Presence Server, as defined by the Internet Engineering Task Force (IETF), the Open Mobile Alliance (OMA), the 3rd Generation Partnership Project (3GPP) and the European Telecommunications Standards Institute (ETSI).

The Presence Server is an entity that accepts, stores and distributes SIP Presence Information.
